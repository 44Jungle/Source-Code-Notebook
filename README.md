# Source-Code-Notebook

关于一些经典论文源码Source Code的中文笔记，尽量做到**整体框架梳理**和**逐行源码注释**。

* 不过太早期的代码一般都模块化了，所以主要更一些后期(2017-NOW)的笔记。

* 语言主要是pytorch和Tensorflow版本的代码，部分keras。

* 有注解错误和思路问题欢迎讨论。
